Set-CMTSStepAutoApplyDriver
---------------------------




### Synopsis
Configure an instance of the Auto Apply Drivers task sequence step.



---


### Description

Use this cmdlet to configure an instance of the Auto Apply Drivers task sequence step.



For more information on this step, see About task sequence steps: Auto Apply Drivers (/mem/configmgr/osd/understand/task-sequence-steps#BKMK_AutoApplyDrivers).



> [!NOTE] > Run Configuration Manager cmdlets from the Configuration Manager site drive, for example `PS XYZ:>`. For more information, see getting started (/powershell/sccm/overview).

### Examples
#### Example 1
```PowerShell
$tsNameOsd = "Default OS deployment"
$tsStepNameAutoApplyDvr = "Auto Apply Drivers"

Set-CMTSStepAutoApplyDriver -TaskSequenceName $tsNameOsd -StepName $tsStepNameAutoApplyDvr -ConsiderAllCategories
```
