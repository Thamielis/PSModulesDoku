Get-ImageQRCode
---------------




### Synopsis
Gets QR code from image



---


### Description

Gets QR code from image



---


### Examples
#### EXAMPLE 1
```PowerShell
Get-ImageQRCode -FilePath "C:\Users\przemyslaw.klys\Downloads\IMG_4644.jpeg"
```
