Get-ChatPayload
---------------




### Synopsis
Get the chat payload.



---


### Description

Get the chat payload as a PSCustomObject.



---


### Examples
#### EXAMPLE 1
```PowerShell
Get-ChatPayload
```



---


### Parameters
#### **AsJson**

Return the chat payload as a JSON string.






|Type      |Required|Position|PipelineInput|
|----------|--------|--------|-------------|
|`[Switch]`|false   |named   |false        |





---


### Syntax
```PowerShell
Get-ChatPayload [-AsJson] [<CommonParameters>]
```
