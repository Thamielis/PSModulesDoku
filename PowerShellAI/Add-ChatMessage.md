Add-ChatMessage
---------------

### Synopsis
Add a chat message to the current chat session.

---

### Description

---

### Examples
#### EXAMPLE 1
```PowerShell
Add-ChatMessage -Message <#PSCustomObject>
```

---

### Parameters
#### **Message**
The chat message to add.

|Type      |Required|Position|PipelineInput|
|----------|--------|--------|-------------|
|`[Object]`|true    |1       |false        |

---

### Syntax
```PowerShell
Add-ChatMessage [-Message] <Object> [<CommonParameters>]
```
